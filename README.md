# AllPayPHPCartModule
歐付寶為各式套裝購物車提供免費金流串接模組，省去複雜的串接程序，安裝檔案即可簡單快速的使用歐付寶金流系統。
● 信用卡一次付清● 信用卡分期(適用於特約商店)● ATM櫃員機● 網路ATM● 超商代碼繳費● 歐付寶帳戶跨境金流● 財付通 Tenpay	
(台灣的店家不須申請財付通帳號，即可收取跨境交易款項，以台幣計價入帳)
